## 게임 시작시 로직
### 1. 앱 실행 시 구글 계정으로 로그인 요청
### 2. 로그인에 성공하면 서버에서 기본 에피소드 1 데이터를 전송
### 3. 사용자가 “시작” 버튼을 누르면 에피소드 메뉴가 열리고 플레이 가능한 에피소드 목록이 표시됨(예: 에피소드 1)
### 4. 에피소드를 완료하면 서버에서 다음 에피소드 데이터를 받아와 계속 진행
